üîÑ How It Works - Step by Step

Step 1: Upload

User drags/drops PDF file on the upload page. File stored in React state.

Step 2: Extract Text

Backend uses PyMuPDF to extract text from all pages of the PDF document.

Step 3: Chunk & Summarize

Text split into 700-word chunks. Each chunk summarized by BART AI model (Facebook).

Step 4: Generate Audio

Summary sent to ElevenLabs API in 200-word chunks. Audio files stitched together.

Step 5: Display Results

User receives text summary + audio player. Can read or listen to content.
